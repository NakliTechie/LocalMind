LocalMind

A private AI chatbot running Gemma entirely in your browser via WebGPU. No server, no API keys, no data leaves your device.

Models are cached after first download — future visits load instantly.

Requirements: Chrome 113+, Edge 113+, or Firefox 130+ with WebGPU.

Powered by Transformers.js + Google Gemma (Apache 2.0).

Available Models

  • Gemma 3 1B (~760 MB) — fast, text-only chat
  • Gemma 4 E2B (~1.5 GB) — text + image + audio + agent
  • Gemma 4 E4B (~4.9 GB) — text + image + audio + agent, best quality

Agent Tools (Gemma 4)

Gemma 4 models have built-in tool calling — math, time, and persistent memory (store & search).

Memory

Memories persist across sessions. Upload .txt/.md files or tell the model to remember things. Click Memory to browse stored memories.

Multimodal (Gemma 4)

  • 📎 Attach — images, audio, or MP4 video
  • 📷 Camera — snap a photo
  • 🎤 Mic — record a voice clip
  • Paste — Ctrl/Cmd+V an image
  • Drag & drop — drop files onto the chat

Video is experimental — keyframes and audio are extracted separately.

Thinking Mode

Open Settings and check “Show reasoning” to see the model’s chain-of-thought.
